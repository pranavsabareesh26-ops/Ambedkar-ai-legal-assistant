# Ambedkar.AI — Indian Legal Assistance Bot

## Overview
Ambedkar.AI is an AI-powered Indian legal assistance chatbot developed to provide accessible legal information through WhatsApp and web platforms.

This project was created as a final-year project focused on simplifying legal guidance using conversational AI workflows and automation systems.
